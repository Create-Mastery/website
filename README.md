## How to run it

| COMMAND         | WHAT IT DOES               |
| --------------- | -------------------------- |
| `npm run dev`   | Start the Astro dev server |
| `npm run build` | Build and review the site  |

> [!WARNING]
> when cloning this repo do this
> ```bash
> npm install
> ```
> then you can run the local server
> ```bash
> npm run dev
> ```
